# ERP Tenants Project

A React-based multi-tenant ERP system built with Vite, TailwindCSS, and Redux Toolkit.

## Features

- Multi-tenant architecture (School, Hospital examples)
- State management with Redux Toolkit
- Routing with React Router DOM
- Styling with TailwindCSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
